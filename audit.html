(4/10)		 CodeCombat uses the guise of a game to teach coding basics, defaulting to python code, with more options available, I find this quite engaging, but not very exciting and I've seen this idea better executed in games like graphomata.

Does it use data to personalize the learning experience? Since students are not required to create an account, students' progress isn't tracked by default. But teachers can select from a list of subjects to be shown, allowing for more personalised learning.
